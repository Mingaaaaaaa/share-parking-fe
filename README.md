本项目为南邮计算机学院大四课设【小区共享停车系统】的前端

**课题内容**

共享停车是为应对城市停车难、停车贵，机动车数量暴增，盘活现有的停车
位资源解决难题最根本的方法。本课题与题目6共用数据库结构及地图相关服务，
要求实现针对小区的共享停车小程序。

**课题要求**

小程序支持2种用户：
1. 管理员（主要对用户的身份、用户车位的使用权
进行验证，停车位导览图管理）；
2. 普通用户。普通用户可具有两种身份：a）车位提供者；b）车位使用者。

对于车位提供者而言，需要提供：
  1. 车位使用权证明的上传功能；
  2. 车位位置的指定；
  3. 车位闲置时间设置功能；
  4. 车位价格设置功能。
    

对于车位使用者而言，需要提供：
  1. 空闲车位的显示；
  2. 空闲车位的指定时间使用权限的购买功能；
  3. 订单取消功能。

**技术栈：Taro + Nest.js + MongoDB**
